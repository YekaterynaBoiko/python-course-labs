# Python Lab Works Repository
This reporsitory contains all Python lab works for the course. 
Each lab works is organized in a separate folder and includes everything needed to complete and review it.

## Lab Work Structure
Each lab folder has the following structures:
```
lab_X/
|__ README.md
|__ requirements.txt
|__ report/
|      |___ answer.md
|__ src/
       |__lab_X.py
```
**Note:** X is the lab number.

## Environment Setup
The environment setup for each lab is decsribed in the lab`s README.md.
