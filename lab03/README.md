# Lab03: Functions as Objects, Lambda, Closures

## Python version
Python 3.10 - this version was used to complete the lab.

## How to run
To run the program, execute:
```
python src/lab03.py
```

## Requiremets 
Any external dependencies used should be listed in
_requirements.txt_.
In this project, no external libraries are used, 
so the file can be empty.

## Description
This lab explores Python functions as first-class
objects. It demonstrates how functions can be 
assigned to variables, passed as arguments, and 
created dynamically. The lab also covers lambda
expressions, closures, and basic functional
compositin, helping to understand core 
functional programming concepts in Python.
