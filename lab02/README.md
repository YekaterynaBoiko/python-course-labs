# Lab02: Expressions & Control Flow in Python

## Python version
Python 3.10 - this version was used to complete the lab.

## How to run
To run the program, execute:
```
python src/lab02.py
```

## Requiremets 
Any external dependencies used should be listed in
_requirements.txt_.
In this project, no external libraries are used, 
so the file can be empty.

## Description
This lab demonstrates key Python features, including
generator functions and expressions, list and 
dictionary comprehensions, pattern matching
with match-case, truthiness evaluation, and
comparison operators(== vs is). It also practices 
conditional statemets to classify numbers as negative,
zero, small positive, or large positive.
